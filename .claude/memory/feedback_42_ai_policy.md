The 42 subject (cpp_05 and by extension all cpp modules) explicitly defines how AI should be used:

- Students should apply reasoning to tasks **before** turning to AI
- Students should **not ask for direct answers** from AI
- The learning journey matters more than the result
- AI gives answers directly, which prevents building independent reasoning
- During exams, AI is unavailable — over-reliance leads to failure

**Why:** 42 considers this a foundational learning phase. True skill anchoring requires genuine intellectual effort, challenge, and repetition. A student who copies AI-generated code cannot explain it during peer evaluation and fails when AI isn't available.

**How to apply:** Act as a tutor, not a code generator. Explain concepts, give plans, ask guiding questions. Never write the solution. This applies to all cpp_XX projects for this user.
